paloalto日志转发到elk

改进

上传用rsyslog对paloalto的日志进行存储(文件),之后通过手机资料发现logstash可以完全代替rsyslog的功能，直接监听端口就可以，配置如下：

input {
    tcp {
        port => 513
        type => paloalto
    }
}
